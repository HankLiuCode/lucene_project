# lucene_project

107-2 高等軟體設計期末專案

主題:利用Django建立一個基於Solr的搜尋引擎，本demo範例為搜尋以爬取下來之新聞文件。

# Demo site : 
http://140.119.19.21:8000/homepage/

**Dependency**
```
python3
Django
pysolr
```
**Usage:**
```
git clone https://github.com/Level1Rookie/lucene_project.git
cd lucene_project
python manage.py runserver <ip>:<port>
```
